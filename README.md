# ✊✋✌️ Stone Paper Scissors Game

A simple **Stone Paper Scissors** game developed in **C** where the user plays against the computer. The computer randomly selects Stone, Paper, or Scissors, and the program determines the winner based on the game rules.

---

## 📌 Features

- 🎮 Play against the computer
- 🎲 Random computer choice using `rand()`
- ✅ Winner is decided automatically
- 🔄 Easy to play in the terminal
- 💻 Beginner-friendly C project

---

## 🛠️ Technologies Used

- C Programming Language
- GCC Compiler
- Standard C Libraries
  - `stdio.h`
  - `stdlib.h`
  - `time.h`

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Stone-Paper-Scissors.git
```

### 2. Navigate to the project folder

```bash
cd Stone-Paper-Scissors
```

### 3. Compile the program

```bash
gcc stone_paper_scissors.c -o game
```

### 4. Run the program

#### Windows

```bash
game
```

#### Linux / macOS

```bash
./game
```

---

## 🎯 Game Rules

- 🪨 Stone beats Scissors
- 📄 Paper beats Stone
- ✂️ Scissors beats Paper
- Same choices result in a Draw
